# 🌱 Eco Defenders: Cyberpunk Runner Pro

A cyberpunk-style eco runner game where players become cyber agents, chase thieves, collect coins, and compete for the highest score.

## 🎮 Features

- 🥷 Choose from 4 cyber heroes
- 🏃 Cyber chase runner gameplay
- 🪙 Collect eco coins
- 🚧 Avoid obstacles
- 🏆 Reward system
- 📊 LocalStorage leaderboard

## 🛠 Technologies

- HTML5
- CSS3
- JavaScript


## 🚀 How to Run

Open the `index.html` file in your browser.



## 📌 Goal

Catch the thief, collect eco rewards, and become the top cyber defender!
